Go Links

User stories

  • User can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • If user pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
  • When user visit that shortened URL, it will redirect user to user's original link.

Usage

GET /api/new/${link} GET /api/${shortLink}
